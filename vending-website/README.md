# Protein Vend — Landing Page Demo

This is a modern, responsive startup landing page for "Protein Vend" — a smart protein vending machine service. It's built with plain HTML, CSS and JavaScript (no frameworks).

Features included:
- Mobile-first responsive design
- Minimal futuristic, fitness-oriented UI with glassmorphism and neon accents
- Smooth scrolling and subtle animations
- Interactive vending machine simulation (brand selection, product selection, add to cart, dispense simulation)
- Clean, well-commented code

Files:
- `index.html` — landing page markup and SVG illustration
- `styles.css` — responsive styles, design system variables and animations
- `script.js` — preloader, navigation and vending simulation logic

Run locally:
1. Open `vending-website/index.html` in a modern browser.
2. Click "Explore Machine" and try the interactive simulation.

Notes:
- This is a frontend-only demo. The dispense and payments are simulated — connect a backend or payment gateway when integrating.
- If you want, I can add a build script, tiny express server for demos, or prepare asset optimization.
